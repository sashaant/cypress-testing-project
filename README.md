# cypress-testing-project


Setup guide:
npm init 
npm install cypress --save-dev
npm install --save-dev typescript


Adding npm scripts:

 "scripts": {
    "cy:open": "cypress open",
    "cy:run": "cypress run"
  }

Now you can invoke the command from your project root like so:
npm run cypress:open
